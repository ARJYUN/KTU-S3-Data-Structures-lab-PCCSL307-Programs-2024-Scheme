## 🚀 Getting Started

### Prerequisites
- GCC Compiler (MinGW for Windows, GCC for Linux/Mac)
- Basic knowledge of C programming

### Compilation & Execution

```bash
# Compile any program
gcc filename.c -o output

# Run the compiled program
./output          # Linux/Mac
output.exe        # Windows
```

### Example
```bash
gcc StackUsingArray.c -o stack
./stack
```
